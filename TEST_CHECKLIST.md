## 테스트 방법

### 로컬 테스트
```bash
npm run dev
```
브라우저에서 `http://localhost:3000` 접속

### 프로덕션 빌드 테스트
```bash
npm run build
npm start
```
브라우저에서 `http://localhost:3000` 접속

### 모바일 테스트
1. Chrome DevTools → Device Toolbar (F12)
2. 다양한 디바이스 크기로 테스트
3. 실제 모바일 기기에서 테스트 (같은 네트워크)
